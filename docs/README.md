# docs/

This directory contains all project documentation for BugLens.

| Subfolder | Contents |
|---|---|
| `srs/` | Software Requirements Specification |
| `sdd/` | Software Design Document |
| `diagrams/` | Architecture and flow diagrams (PNG, draw.io, PlantUML) |
| `api/` | API documentation (OpenAPI/Swagger exports, Postman collections) |
| `testcases/` | Test case documents and test plans |

Documentation is written in Markdown unless otherwise noted. It is updated progressively as each development phase is completed.
