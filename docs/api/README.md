# docs/api/

**API Documentation**

This folder contains API documentation for BugLens services.

## Contents

- `openapi.yaml` — OpenAPI 3.0 spec for Backend API (auto-exported from Swagger)
- `postman_collection.json` — Postman collection for manual API testing
- Endpoint reference documents (Markdown)

## Live Docs

When services are running locally, interactive docs are available at:

| Service | URL |
|---|---|
| Backend API (Swagger) | http://localhost:3000/api/docs |
| AI Service (FastAPI) | http://localhost:8000/docs |

> 📝 API spec files are exported and committed during **Phase 6 — Documentation**.
