# docs/diagrams/

**Architecture and Flow Diagrams**

This folder contains visual diagrams for the BugLens system, including:

- System architecture diagram (component overview)
- MVP data flow diagram (end-to-end bug report journey)
- Database Entity-Relationship (ER) diagram
- Sequence diagrams for key workflows
- Deployment diagram

## Formats

- `.drawio` — editable diagrams (draw.io / diagrams.net)
- `.png` / `.svg` — exported images for documentation
- `.puml` — PlantUML source files

> 📝 Diagrams are added progressively as each phase is implemented.
