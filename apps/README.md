# apps/

This directory contains all **frontend** applications in the BugLens monorepo.

| App | Description |
|---|---|
| `dashboard-web/` | React + Vite dashboard for developers to manage and review bug reports |
| `browser-extension/` | Chrome Extension (Manifest V3) for capturing and submitting bug reports |
| `demo-target-web/` | Sample web application used to demonstrate bug reporting in action |

Each app is self-contained with its own dependencies and can be run independently.
